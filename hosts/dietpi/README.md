# DietPI

DietPI configuration

## Services

**Backups**
Backup data to drive and nixos

**DNS & DHCP** Pi-hole and dnsmasq

**Dynamic DNS** ddclient

**Routing** nftables

Forward internet to my domain (treywilkinson.com) to my pi.

**VPN** WireGuard

**SSH** Dropbear

## Hardware

Raspberry Pi 4
Gigabit ethernet
